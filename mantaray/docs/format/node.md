# node binary format

The following describes the format of a node binary format.

```
┌──────────────────────────────┐
│  magic(0x2D833CCB) <4 byte>  │
├──────────────────────────────┤
│    version(1) <1 byte>       │
├──────────────────────────────┤
│    refBytesSize <1 byte>     │
├──────────────────────────────┤
│     padding <26 byte>        │
├──────────────────────────────┤
│       nonce <32 byte>        │
├──────────────────────────────┤
│       entry <32 byte>        │
├──────────────────────────────┤
│  forksIndexBytes <32 byte>   │
├──────────────────────────────┤
│ ┌──────────────────────────┐ │
│ │          Fork 1          │ │
│ ├──────────────────────────┤ │
│ │          ...             │ │
│ ├──────────────────────────┤ │
│ │          Fork N          │ │
│ └──────────────────────────┘ │
├──────────────────────────────┤
│    forksNodeTypes <bytes>    │
└──────────────────────────────┘
```

## Fork

```
┌───────────────────────────┬────────────────────────┬──────────────────┬──────────────────────┐

└───────────────────────────┴────────────────────────┴──────────────────┴──────────────────────┘
```
